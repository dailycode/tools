O front end é apresentação visual de um site. Desenvolvedores enfrentam bastantes obstáculos ao escrever código. Para amenizar a situação deve-se fazer o uso de algumas ferramentas, para que possa facilitar o processo do desenvolvimento e o tornar mais produtivo no seu fluxo de trabalho. O projeto Tools front end elaborou uma pesquisa com 100 Frontenders para saber qual as ferramentas que costumam usar diariamente.
